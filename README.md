# step-io

STEP (ISO 10303) file I/O for Rust.

⚠️ **Very early alpha.** Public API is unstable and may change at any time.

## Scope

The current focus is AP203 / AP214 (CD, DIS, IS) / AP242 DIS 3D CAD
geometry and assembly.
